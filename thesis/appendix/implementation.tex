\section{Building the PDF}

From the repository root on Windows PowerShell:

\begin{verbatim}
cd thesis
pdflatex main.tex
bibtex main
pdflatex main.tex
pdflatex main.tex
\end{verbatim}

Or run \texttt{scripts/compile\_thesis.ps1} which executes the same sequence.

\section{Training Commands}

Encode baseline logs:

\begin{verbatim}
python scripts/encode_logs.py logs/baseline_*.jsonl
\end{verbatim}

Train BC TurnTransformer:

\begin{verbatim}
python scripts/train_bc.py logs/encoded_*.jsonl --epochs 30
\end{verbatim}

Train belief-augmented model when encoded records include belief labels:

\begin{verbatim}
python scripts/train_belief_bc.py logs/encoded_*.jsonl --epochs 30
\end{verbatim}

\section{TurnTransformer Forward Pass}

Given token IDs \texttt{token\_ids} of shape $(B, T, 13)$ and turn mask
\texttt{turn\_mask} of shape $(B, T)$, the model applies slot self-attention
per turn, causal turn attention across history, and linear action logits over
the final valid turn embedding.

BeliefTransformer adds per-slot species heads and optional fusion into the
policy input when enabled.
